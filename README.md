This is an implementation of the Open Contracting Data Standard

### Installing

npm install

node app.js

### carep

Get a contract

```
GET /contract/ocds-k50g02-02-2018
```

Get all contracts

```
GET /contract
```

Sign up

```
POST /signup
```

Login

```
POST /login
```

Register Company

```
POST /register-company
```

Upload contracts file .xlsx/.xls

```
POST /upload-contracts
```
